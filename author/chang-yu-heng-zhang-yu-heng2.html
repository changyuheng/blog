Sat 24 November 2012

  1. 什麼是 Codeforces？

    Codeforces 是一個知名的程式競賽 (programming contest) 網站。我們可以在此網站上討論程式設計，練習或比賽解題，也可以舉辦比賽。

  2. 支援哪些語言？

    • C
    • C++
    • C#
    • Delphi
    • Free Pascal
    • Haskell
    • Java
    • OCaml
    • PHP
    • Python
    • Ruby
    • Scala

  3. 怎麼開始？

    1. 註冊

      1. 進入 Codeforces 首頁

      2. 點選右上角「Register」以進入註冊頁面

      3. 填寫註冊表單，完成後按下「Register (確定註冊)」

      經過以上步驟即可完成註冊。
